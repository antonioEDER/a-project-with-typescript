<h2 align="center">Install</h2>

Build Development with yarn:

```bash
yarn build:dev
```

Build Prod with yarn:

```bash
yarn build
```

Start with yarn:

```tsc
yarn tsc
```

```bash
yarn start
```

Serve with webpack-dev-serve:

```bash
yarn serve
```

Serve with express Development:

```bash
yarn express
```

Serve with express Prod:

```bash
yarn express:prod
```

Serve with Hot Module Replacemnt:

```bash
yarn hot
```

Clean path /dist:

```bash
yarn clean
```
